プライバシーポリシー

最終更新日: 2026年3月5日

ClipMontage（以下「本サービス」）は、ブラウザ上で動作する AI 動画編集ツールです。本プライバシーポリシーは、本サービスが収集する情報およびその取り扱いについて説明します。

1. 収集する情報

  • アカウント情報: Google サインインまたはメール/パスワードによる認証を利用した場合、Firebase Authentication を通じてメールアドレスおよびプロフィール情報（表示名・アバター URL）を取得します。
  • 動画ファイル: アップロードされた動画はお使いのブラウザ内でのみ処理されます。弊社サーバーには送信・保存されません。
  • AI プロバイダー API キー: 入力された API キーはブラウザの sessionStorage にのみ保存され、セッション終了時に自動削除されます。弊社には送信されません。
  • 利用ログ: Firebase によるアクセスログ（IP アドレス・リファラー）が収集される場合があります。

2. 情報の利用目的

  • ユーザー認証およびアカウント管理
  • 本サービスの提供・改善
  • 不正アクセスの検出・防止

3. 第三者への提供

本サービスは以下の外部サービスを利用します。各サービスのプライバシーポリシーをご確認ください。

  • Firebase (Google LLC): 認証・ホスティング — Google プライバシーポリシー
  • Groq / Gemini / HuggingFace: ユーザーが選択した場合、フレーム画像データが各サービスに送信されます。利用前に各サービスの規約をご確認ください。

上記以外の第三者に個人情報を販売・提供することはありません。

4. データの保存期間

Firebase Authentication に保存されたアカウント情報は、アカウント削除時まで保持されます。動画データおよび API キーはブラウザ外に保存されません。

5. ユーザーの権利

  • アカウント情報の確認・修正
  • アカウントの削除（サービス内の設定またはお問い合わせにより対応）
  • データポータビリティの要求

6. Cookie

本サービスは Firebase Authentication のセッション維持のために Cookie を使用します。ブラウザ設定により Cookie を無効化できますが、認証機能が利用できなくなる場合があります。

7. 未成年者のプライバシー

本サービスは 13 歳未満の方を対象としていません。13 歳未満の方の個人情報を意図的に収集することはありません。

8. 本ポリシーの変更

本ポリシーを変更する場合、本ページにて最新版を公開します。重要な変更がある場合は、サービス内でお知らせします。

9. お問い合わせ

本ポリシーに関するご質問は、利用規約のお問い合わせ先までご連絡ください。
